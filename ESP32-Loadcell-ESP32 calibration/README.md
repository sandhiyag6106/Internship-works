Overview- Interfacing 5Kg loadcell with ESP32 S3 devkit for calibrating and measuring weight of objects.

Components used - 5Kg Loadcell, ESP32 S3 devkit, HX711 amplifier module, breadboard, connecting wires.

Software used - Arduino IDE.

Project Insights - During the project the challenges faced were unstable readings and sensor noise.
I addressed these by performing multi-point calibration, improvising and testing the code with known weights.
Through the project I gained practical exposure to sensor calibration techniques, ADC based measurement and hands-on debugging.
