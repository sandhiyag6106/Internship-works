Overview - Interfacing a 4G LTE SIM communication module with ESP32 S3 to initialize and detect SIM card.
Components used - ESP32 S3 dev module,  A7670C 4G LTE module, SIM Card, Connecting wires.
Software used - Arduino IDE.
Project Insights - Faced issues regarding SIM card not detected and serial communication produced garbage value in Serial monitor.
The problems were resolved by verifying baud rate, AT command response and improving power stability.
Through the work gained practical exposure in UART communication, serial debugging and cellular module interfacing.
