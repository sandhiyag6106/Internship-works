Overview - Interfacing an OLED with ESP32 S3 dev module to display text and messages using I2C communication.
Components used - ESP32 S3 devkit, OLED, Breadboard, connecting wires.
Software used - Arduino IDE.
Project Insights - Initial issue was display not detected and resolved the problem using I2C scanning testing and correcting thr address configuration.
Through the process gained hands-on experience in I2C communication, display interfacing, and embedded UI interface.
